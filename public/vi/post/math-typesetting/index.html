Định Dạng Toán Học

  • 8/3/2019
  • Đọc trong 1 phút

Công thức toán học trong một dự án Hugo có thể được bật lên bằng cách sử dụng các thư viện JavaScript của bên thứ ba.

Trong ví dụ này, chúng ta sẽ sử dụng KaTeX

  • Tạo một partial dưới /layouts/partials/math.html
  • Trong partial này, tham chiếu đến Tiện ích Tự động render hoặc tải các tập tin script này về máy.
  • Bao gồm partial trong các mẫu của bạn như sau:
{{ if or .Params.math .Site.Params.math }}
{{ partial "math.html" . }}
{{ end }}


- To enable KaTex globally set the parameter `math` to `true` in a project's configuration
- To enable KaTex on a per page basis include the parameter `math: true` in content files

**Note:** Use the online reference of [Supported TeX Functions](https://katex.org/docs/supported.html)



### Examples

Inline math: \\(\varphi = \dfrac{1+\sqrt5}{2}= 1.6180339887…\\)

Block math:

$$
 \varphi = 1+\frac{1} {1+\frac{1} {1+\frac{1} {1+\cdots} } }
$$
